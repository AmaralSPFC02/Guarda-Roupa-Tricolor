Guarda-Roupa Tricolor

Projeto final desenvolvido para a disciplina XDES03 (Programação Web) da UNIFEI.
O sistema é um catálogo para gerenciamento de camisas entre 2010-2025 do São Paulo FC.

Não sabia que precisaria ser subido antes da apresentação, achei que era só pra mandar depois, não achei que teria utilidade visto que fiz sozinho então os commits não fariam diferença pra ver quem fez o que
Fiz tudo no VSCode e com uso de terminais, que rodam tranquilamente um projeto de Web
Criei esse repositório na pressa uns 45 min antes da apresentação após conversar com gente do meu projeto que apresentou antes, e me disseram que o Sr. pedia o ReadMe nas apresentações

Tecnologias Utilizadas:

- Frontend: React + Vite
- Backend :Node.js + Express
- Persistência: File System (JSON)
- Estilização: CSS Puro (Grid e Flexbox)
- API Externa: Open-Meteo (Previsão do tempo)
